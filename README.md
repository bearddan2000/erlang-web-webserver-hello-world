# erlang-web-webserver-hello-world

## Description
A POC erlang webserver

## Tech stack
- bash
- erlang 10.7.1

## Docker stack
- erlang:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
